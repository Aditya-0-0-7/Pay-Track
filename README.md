Main Application Module

The main application has two important components:
 
![image](https://user-images.githubusercontent.com/105495413/203132360-78927a17-7c1e-4975-99f9-bd5909120082.png)

•	View Transactions
 
![image](https://user-images.githubusercontent.com/105495413/203132486-29ccb2b5-11d2-425c-99a9-10d9816af558.png) 
 
This module allows users to view every transaction in which they are involved. It has three options-initiated Transaction, Pending Transaction and Confirmed Transaction which allow users to view transaction under different Category.

Initiated Transaction- It includes those transactions which are created by the user himself. That is in all those transactions in which user act as lander.

Pending Transaction- It includes those transactions in which user act as borrower and had to accept the request created by lander to commit the transaction.

Confirmed Transaction- It includes those transactions whose request has been accepted and being commit into block chain.

•	Create Transactions
 
![image](https://user-images.githubusercontent.com/105495413/203132547-0d544ecb-fd58-428d-9239-0c19580d5445.png)

This application module allows landers to create transaction request to borrowers.
